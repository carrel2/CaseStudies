\Doctrine\Common\Collections

\Doctrine\Common\CollectionsArrayCollection

An ArrayCollection is a Collection implementation that wraps a regular PHP array.

Warning: Using (un-)serialize() on a collection is not a supported use-case and may break when we change the internals in the future. If you need to serialize a collection use \Doctrine\Common\Collections\toArray() and reconstruct the collection manually.
